# BHCC Spring 2026 — Assignment Repository

This repository contains coursework and assignments for the Spring 2026 semester at Bunker Hill Community College.

## Courses

### JavaScript
Client-side scripting and dynamic web development. Covers core JavaScript fundamentals including DOM manipulation, event handling, and modern ES6+ syntax.

### JSON
Data interchange using JavaScript Object Notation. Focuses on JSON structure, parsing, validation, and its role in web APIs and data-driven applications.

### Mobile
Mobile application development. Covers building applications for mobile platforms, responsive design principles, and mobile-specific UI/UX considerations.

### Node.js
Server-side JavaScript using the Node.js runtime. Covers building backend services, working with npm, file system operations, and server frameworks.

## Repository Structure

```
assignmentrepo/
├── javascript/
├── json/
├── mobile/
├── nodejs/
└── README.md
```

Each directory corresponds to a course and will contain that course's submitted assignments organized throughout the semester.
